# hubot-yrno

Weather information for Norway

See [`src/yrno.coffee`](src/yrno.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-yrno --save`

Then add **hubot-yrno** to your `external-scripts.json`:

```json
[
  "hubot-yrno"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
